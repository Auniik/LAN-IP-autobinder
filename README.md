
# LAN IP Autobinder

This script will be watching your LAN IP. If any change detected it will trigger and bind the IP with a localdomain (default: localws)

#### Supports: 
- NetworkManager


### > CAUTION
 Make a backup your /etc/hosts



#### Installation:
```sh
sudo sh ./install.sh
```

#### Output:
Check output in your /etc/hosts file
